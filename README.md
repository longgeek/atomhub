# The AtomHub website

This repo houses the assets used to build the AtomHub website, available demo at http://test.openatom.cn/atomhub/#/repos.

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
